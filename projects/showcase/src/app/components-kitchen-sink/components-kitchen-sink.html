Buttons

Normal: intent omitted (default) for routine, non-destructive actions.

Primary: intent="primary" for the highest-emphasis action in a view, usually one per section.

Secondary: intent="secondary" for alternate actions when primary already represents the main path.

Quiet: intent="quiet" for low-emphasis actions that remain discoverable without stealing focus.

Danger: intent="danger" for destructive actions; pair with clear labeling and confirmation when irreversible.

Icon action buttons

Icon-only: use shape="round" size="sm" and always provide an accessible name via aria-label. Keep SVGs decorative with aria-hidden="true".
